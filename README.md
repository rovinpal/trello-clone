# Trello Clone

A Trello-inspired task management application built using vanilla JavaScript, HTML, and CSS, powered by the official Trello API.

This project demonstrates DOM manipulation, asynchronous API integration, and CRUD operations while replicating core Trello functionality.

---

## Features

### Board Management

* Display all lists and their corresponding cards within a Trello board.

### List Management

* View all lists in a board.
* Create new lists.
* Rename existing lists.
* Delete lists.

### Card Management

* Display all cards within a list.
* Create new cards.
* Rename cards.
* Archive cards.

### Checklist Management

* Display all checklists within a card.
* Create new checklists.
* Rename checklist groups.
* Delete checklists.

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Trello REST API
* DOM Manipulation
* Fetch API
